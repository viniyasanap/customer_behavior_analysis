# customer_behavior_analysis
data anaylitics project showcasing behavior analysis using pythons,sql and power Bi
Overview

This project demonstrates a complete data analytics workflow, starting from raw data and ending with an interactive dashboard and presentation. It includes data cleaning, exploratory data analysis (EDA), SQL analysis, dashboard creation in Power BI, and reporting to generate meaningful business insights.

Dataset

The project uses a structured dataset containing records for analysis.

The dataset was used to:

Explore data patterns
Clean missing and inconsistent values
Perform business analysis
Build visualizations and dashboards

Project Workflow
1. Data Loading
Imported the dataset into Python using Pandas.
Examined the dataset structure and data types.
2. Exploratory Data Analysis (EDA)
Generated summary statistics.
Identified trends and patterns.
Visualized data using charts and graphs.
3. Data Cleaning
Handled missing values.
Removed duplicate records.
Corrected inconsistent data.
Converted columns to appropriate data types.
4. SQL Analysis
Imported cleaned data into a SQL database.
Executed SQL queries to answer business questions.
Used filtering, grouping, joins, sorting, and aggregate functions.
5. Power BI Dashboard
Connected data to Power BI.
Built interactive dashboards.
Created KPIs and visual reports.
Added filters and slicers for better analysis.
6. Report & Presentation
Prepared a project report summarizing the analysis.
Created a professional presentation using Gamma.
